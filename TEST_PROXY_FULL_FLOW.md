# ✅ ПОЛНЫЙ ТЕСТ: Система управления прокси

## 🎯 Полный цикл работы с прокси

### ЭТАП 1: Генерация прокси админом

1. **Админка** → `http://localhost:8080/admin/login`
   - Логин: `admin@streamer.local / admin123`

2. **Пользователи** → нажми **✏️** (редактировать) на User 1

3. **Прокрути вниз** → секция "Управление прокси"

4. **Выбери формат:**
   - TXT (IP:PORT:USER:PASS)
   - CSV (таблица)
   - JSON (объекты)

5. **Нажми** "Создать и скачать"

6. **Файл скачается:** `proxies_user_2_2026-03-09_172534.txt` (или .csv/.json)

**Пример содержимого (TXT):**

### ЭТАП 2: Пользователь загружает прокси

1. **Пользователь** → `http://localhost:8080/login`
   - Логин: `user1@streamer.local / user12345`

2. **Перейди** на страницу **"Мои прокси"**

3. **Нажми** "Загрузить Список"

4. **Выбери** скачанный файл (от админа)

5. **Нажми** "Загрузить"

6. **Результат:** "Добавлено: 11, пропущено дубликатов: 0"

7. **ВАЖНО:** Страница останется **ПУСТОЙ** → прокси ещё не активированы!

---

### ЭТАП 3: Админ активирует прокси

**Вариант A: Через страницу пользователя**

1. **Админка** → Пользователи → ✏️ User 1

2. **Нажми** "Просмотр прокси пользователя (11)"

3. **Увидишь таблицу** со всеми 11 прокси:
   - IP адрес
   - Порт  
   - Логин
   - Пароль (первые 4 символа + ••••)
   - **Статус** (dropdown с выбором)
   - Дата
   - Действия (🗑️ удалить)

4. **Измени статус** через dropdown:
   - Выбери "✅ Активный" → статус сохранится автоматически
   - Или выбери "🟢 Онлайн" / "🔴 Офлайн"

5. **Кнопка "Удалить все прокси"** (красная, вверху справа)

**Вариант B: Через общий список прокси**

1. **Админка** → **Прокси** (в меню слева)

2. **Фильтр** → выбери User 1

3. **Нажми** "Активировать все pending"

4. Все прокси получат статус "Активный"

---

### ЭТАП 4: Пользователь видит прокси

1. **Вернись** в личный кабинет: `http://localhost:8080/dashboard/proxies`

2. **Обнови страницу** (F5)

3. **Теперь видишь таблицу** с прокси:
   - IP адрес
   - Порт
   - Логин
   - Статус (🟢/🟣/🔴)
   - Дата
   - Кнопка 🗑️ (удалить)

4. **Summary показывает:**
   ```
   Всего: 11 | Онлайн: 3 | Офлайн: 2
   ```

## 🧪 Тест через curl

```bash
# 1. Логин
curl -X POST http://localhost:8080/api/login \
  -H "Content-Type: application/json" \
  -d '{"email":"user1@streamer.local","password":"user12345"}' \
  -c cookies.txt

# 2. Загрузить TXT
curl -X POST http://localhost:8080/api/proxies/upload \
  -b cookies.txt \
  -F "file=@proxies-example.txt" | jq

# 3. Проверить (пусто - pending)
curl -s http://localhost:8080/api/proxies -b cookies.txt | jq

# 4. Активировать (через tinker)
docker compose exec -T backend php artisan tinker \
  --execute="Proxy::where('user_id', 2)->update(['status' => 'active']); echo 'Done';"

# 5. Проверить снова (теперь видны!)
curl -s http://localhost:8080/api/proxies -b cookies.txt | jq

# 6. Удалить прокси
curl -X DELETE http://localhost:8080/api/proxies/1 \
  -b cookies.txt | jq
```
