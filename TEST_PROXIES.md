## 📊 ЧАСТЬ 5: Проверка через curl

```bash
# 1. Получить список прокси
curl -s http://localhost:8080/api/proxies -b cookies.txt | jq

# 2. Загрузить TXT
curl -X POST http://localhost:8080/api/proxies/upload \
  -b cookies.txt \
  -F "file=@proxies-example.txt" | jq

# 3. Загрузить JSON
curl -X POST http://localhost:8080/api/proxies/upload \
  -b cookies.txt \
  -F "file=@proxies-example.json" | jq

# 4. Загрузить CSV
curl -X POST http://localhost:8080/api/proxies/upload \
  -b cookies.txt \
  -F "file=@proxies-example.csv" | jq

# 5. Удалить прокси (замени {id})
curl -X DELETE http://localhost:8080/api/proxies/1 \
  -b cookies.txt | jq
```

## ✨ Что должно работать

### ✅ Пользователь
- Загрузка файлов (TXT, CSV, JSON)
- Просмотр активных прокси
- Удаление своих прокси
- Статистика (всего, онлайн, офлайн)

### ✅ Администратор
- Просмотр всех прокси всех пользователей
- Фильтрация по пользователю и статусу
- Активация/деактивация прокси
- Массовая активация pending
- Удаление прокси

### ✅ Система
- Уникальность по IP адресу
- Поддержка 3 форматов файлов
- Парсинг 3 форматов строк (TXT)
- Статусы: pending, active, online, offline
- Ограничение: до 1000 прокси/юзер
