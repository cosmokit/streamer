# ✅ Тестирование генератора прокси в админке

## Функционал

В админке, на странице **редактирования пользователя**, есть кнопка **"Создать и скачать прокси"**.

При нажатии:
1. Генерируется **11 уникальных прокси**
2. Файл автоматически скачивается: `proxies_user_{id}_{timestamp}.txt`
3. Формат: `IP:PORT:USERNAME:PASSWORD` (по одному на строку)

## Где найти кнопку

1. Открой: `http://localhost:8080/admin/login`
2. Логин: `admin@streamer.local / admin123`
3. Перейди: **Пользователи** → нажми ✏️ на любом пользователе
4. Прокрути вниз → увидишь карточку **"Генератор прокси"**
5. Нажми кнопку **"Создать и скачать прокси"**
6. Файл скачается автоматически: `proxies_user_2_2026-03-09_151523.txt`

## Особенности

- ✅ Все IP уникальные (проверка в цикле)
- ✅ Реалистичные IP из популярных дата-центров
- ✅ Стандартные прокси-порты
- ✅ Разнообразные логины с суффиксами
- ✅ Сложные пароли (буквы + цифры + спецсимволы)
- ✅ Формат совместим с загрузкой: `IP:PORT:USER:PASS`
