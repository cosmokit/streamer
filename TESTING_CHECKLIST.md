## 🧪 Инструкция для тестирования

### Шаг 1: Регистрация и вход

1. Открой http://localhost/register
2. Зарегистрируй нового пользователя:
   - Email: `test_YOUR_NAME@example.com`
   - Password: `Test123456!`
3. После регистрации должен быть автоматический вход
4. Проверь что попал на страницу /dashboard/progress

### Шаг 2: Прогресс обучения

1. На странице /dashboard/progress должны быть 4 шага:
   - Шаг 1: Загрузка и настройка стримовых ключей
   - Шаг 2: Создание плейлистов
   - Шаг 3: Настройка расписания стримов
   - Шаг 4: Итоговая настройка и запуск
2. Нажми "Начать" на первом шаге
3. Нажми "Отправить на проверку"
4. Статус должен измениться на "На проверке"

### Шаг 3: Страница Traffic

1. Перейди на /dashboard/traffic
2. Должна быть кнопка "Запустить стрим"
3. НЕ нажимай её пока (нужны прокси для теста)

### Шаг 4: Загрузка прокси

1. Создай файл `proxies.txt` с содержимым:
   ```
   1.2.3.4:8080:user1:pass1
   2.3.4.5:8080:user2:pass2
   3.4.5.6:8080:user3:pass3
   ```
2. Перейди на /dashboard/profile
3. Проверь:
   - ❌ НЕТ поля Email
   - ✅ ЕСТЬ поле Username
4. Загрузи файл proxies.txt
5. Проверь что загрузка прошла успешно

### Шаг 5: Другие страницы пользователя

Проверь что открываются:
- /dashboard/videos
- /dashboard/templates
- /dashboard/help

### Шаг 6: Выход и вход в админку

1. Нажми "Выход"
2. Перейди на /admin/login
3. Войди:
   - Email: admin@example.com
   - Password: admin123

### Шаг 7: Список пользователей

1. Перейди на /admin/users
2. Проверь:
   - ✅ Есть фильтр по дням стримов
   - ✅ Есть поиск
3. Найди своего зарегистрированного пользователя через поиск
4. Кликни на него

### Шаг 8: Профиль пользователя в админке

1. Проверь что видны загруженные прокси
2. Проверь кнопки:
   - "Активировать все"
   - "Деактивировать все"
3. Проверь что можно скачать файл прокси

### Шаг 9: Прогресс пользователя

1. Перейди на вкладку "Прогресс обучения"
2. Должен быть виден шаг который отправлен на проверку
3. Нажми "Подтвердить"
4. Статус должен измениться

### Шаг 10: Генерация пользователя

1. Перейди на /admin/users/generate
2. Заполни:
   - Telegram: `@testuser_YOUR_NAME`
   - Twitch: `testchannel_YOUR_NAME`
3. Нажми "Сгенерировать"
4. Должен появиться блок с:
   - ✅ Логином
   - ✅ Паролем
   - ✅ Ссылкой на Twitch
   - ✅ Telegram handle

### Шаг 11: Генерация прокси

1. Перейди на /admin/proxies/generate
2. Проверь поля:
   - ✅ Количество
   - ✅ Статус (pending/active/failed)
   - ❌ НЕТ кнопки "Скачать"
3. Сгенерируй 5 прокси со статусом "active"

### Шаг 12: Impersonation

1. Вернись на /admin/users
2. Найди кнопку "Войти как пользователь" у любого юзера
3. Кликни
4. Должен попасть в кабинет пользователя
5. Проверь что есть кнопка "Вернуться в админку"
6. Кликни - должен вернуться в админку

### Шаг 13: Управление стримом

1. Войди под пользователем (через impersonation или обычный вход)
2. Перейди на /dashboard/traffic
3. Если нет 65 активных USA прокси:
   - При клике на "Запустить 4-й стрим" должен появиться попап с сообщением
4. Если есть прокси или это не 4-й стрим:
   - Кликни "Запустить стрим"
   - Текст должен измениться на "Завершить стрим"
   - Должно показаться "День 1 из 4"
   - Кликни "Завершить стрим"

### Шаг 14: Подписки

1. Проверь страницу профиля
2. Должен быть блок "Подписка LITE"
3. Должна быть кнопка "Перейти на подписку PRO"

### Шаг 15: Редактирование помощи

1. В админке перейди на /admin/help
2. Кликни на любую статью
3. Проверь что можно редактировать
4. Сохрани изменения
